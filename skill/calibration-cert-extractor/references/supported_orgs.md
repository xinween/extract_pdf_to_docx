## 如何识别证书机构

查看 PDF 第一页：

1. **机构名标识**：封面顶部或底部通常有机构全称
2. **证书编号前缀**：
   - `HDYL` / `HDWD` → 邦盟
   - `AIMT` → 爱准
   - 字母+5位数字+字母+5位数字 → 上海质检院
3. **CNAS/CMA 章**：章上的机构缩写

## OCR 注意事项

1. 扫描版 PDF 可能存在：
   - 文字模糊
   - 表格线缺失
   - 印章覆盖部分文字

2. 处理建议：
   - 使用 PaddleOCR 逐页扫描（DPI=200，必要时提升到300）
   - 必须逐页读取，不跳页
   - 封面判断用关键词组合（编号+仪器名+机构名），不依赖"第1页共N页"
   - 必要时手动核对关键字段
